# Murfreesboro Home Pros homepage prototype

Static HTML/CSS homepage prototype using the supplied SVG logo and current brand direction.

Open `index.html` directly or serve locally:

```bash
python3 -m http.server 8080
```

Core positioning: local home-service research + screened one-at-a-time referrals. Avoids contractor/GC language and includes referral-service disclaimers.
